SymptomLog 🌿

A sleek, privacy-focused health tracking web app. It allows users to log symptoms, track intensity, and analyze patterns—all while keeping data 100% local and secure.

 ✨ Features
- **Privacy-First**: Data is stored locally on the user's device (no server-side tracking).
- **Pattern Insights**: Built-in logic to correlate sleep, stress, and symptom severity.
- **Modern UI**: Fully responsive, glassmorphic design with a seamless Dark Mode.
- **Doctor-Ready Export**: One-click export to copy logs for medical appointments.
